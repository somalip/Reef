Settings

Execution

Control how Reef interacts with page elements.

Action mode Determines whether Reef can execute clicks and form submissions or only navigate to links.
Destructive action confirmation Require explicit confirmation before executing destructive actions (delete, remove, checkout, etc).
Auto-execute on selection Automatically execute an action when selected from search results, instead of requiring a second click.

Privacy & Security

Control what Reef can access and index.

Exclusion selectors CSS selectors for elements Reef must never index or interact with. Sensitive inputs (passwords, payment, SSN) are always excluded.
Blocked domains Domains where Reef will not activate. One per line.

Search

Configure search behavior and result display.

Default results per page Number of results shown before pagination.
Scoring algorithm Algorithm used to rank search results.
Fuzzy matching Enable typo-tolerant search with Levenshtein distance.
Result diversity (MMR) Diversify results using Maximal Marginal Relevance to avoid redundant entries.
Cross-tab indexing Index pages from the same origin across multiple open tabs.

Appearance

Customize the look and feel of the extension UI.

Theme Choose a color theme for the popup and settings pages. System will follow your OS preference.
Compact mode Reduce spacing and font sizes for a denser information layout.

Data Management

Manage cached indices and stored data.

Cached indices Clear all cached page indices. Pages will be re-indexed on next visit.
Query history Clear all tracked search queries and popularity data.
Reset all settings Restore all settings to their default values. This cannot be undone.
