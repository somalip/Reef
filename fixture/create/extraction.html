Overview

The extraction module provides functions for extracting content from HTML documents. It handles sections (headings), actions (buttons), fields (form inputs), links, files, media, and JSON-LD structured data. 

/**
 * @file Content extraction functions for indexing web pages.
 * Extracts sections, actions, fields, links, files, media, and structured data from HTML.
 */

import { IndexRecord } from './types.js';

// Main extraction functions
export function stripTags(value: string): string
export function generateSelector(element: Element): string

// Content extraction
export function extractSections(html: string, url: string): IndexRecord[]
export function extractActions(html: string, url: string, excludeSelectors?: string): IndexRecord[]
export function extractFields(html: string, url: string): IndexRecord[]
export function extractLinks(html: string, url: string): IndexRecord[]
export function extractFiles(html: string, url: string, extensions?: string): IndexRecord[]
export function extractMedia(html: string, url: string): IndexRecord[]
export function extractStructuredData(html: string, url: string): IndexRecord[]

// Utility
export function extractHiddenContent(doc: Document): void
export function resolveUrl(value: string, base: string): string

extractSections()

Extracts heading elements (h1-h6) and their associated content from HTML. Creates a structured representation with breadcrumbs and body text for each section.

Signature

function extractSections(html: string, url: string): IndexRecord[]

// Returns an array of IndexRecord objects with type: 'section'

Parameters

Parameter Type Description
html string The HTML content to parse
url string The page URL for creating fragment URLs

How It Works

// 1. Clean HTML by removing scripts, styles, comments
const cleanHtml = html
  .replace(/<script[\s\S]*?<\/script>/gi, ' ')
  .replace(/<style[\s\S]*?<\/style>/gi, ' ')
  .replace(/<!--[\s\S]*?-->/g, ' ');

// 2. Find all headings with regex
const headingRegex = /<(h[1-6])[^>]*>([\s\S]*?)<\/h[1-6]>/gi;

// 3. For each heading, extract content until next heading
// 4. Generate breadcrumb trail from all preceding headings
// 5. Create selector based on heading ID or parent section

sections[i] = {
  id: `${url}#${heading.id}`,
  url: `${url}#${heading.id}`,
  headingText: heading.text,
  headingId: heading.id,
  breadcrumb: 'Parent › Current › Heading',
  bodyText: 'Content between headings...',
  type: 'section',
  selector: '#heading-id' // or parent section selector
};

extractActions()

Extracts clickable elements (buttons, links with button roles) that can trigger actions. Identifies destructive actions like delete, remove, and submit.

Signature

function extractActions(
  html: string,
  url: string,
  excludeSelectors?: string
): IndexRecord[]

// Returns an array of IndexRecord objects with type: 'action'
// Each record includes: selector, label, destructive (boolean)

Target Elements

// Elements targeted by extractActions():
const selectors = [
  'button',                    // <button> elements
  '[role="button"]',          // Elements with button role
  'input[type="button"]',     // Button inputs
  'input[type="submit"]',     // Submit inputs
  'summary',                  // Details summaries
  '[data-reef-action]'       // Custom action markers
];

Label Extraction Priority

// extractActionName() priority:
// 1. aria-label attribute
// 2. aria-labelledby reference
// 3. textContent (visible text)
// 4. title attribute

function extractActionName(element) {
  const ariaLabel = element.getAttribute('aria-label');
  if (ariaLabel?.trim()) return ariaLabel.trim;
  
  const ariaLabelledBy = element.getAttribute('aria-labelledby');
  if (ariaLabelledBy) {
    const labelledElement = document.getElementById(ariaLabelledBy);
    if (labelledElement?.textContent?.trim()) {
      return labelledElement.textContent.trim();
    }
  }
  
  const textContent = element.textContent?.trim();
  if (textContent) return textContent;
  
  const title = element.getAttribute('title');
  if (title?.trim()) return title.trim();
  
  return null;
}

Destructive Actions: Actions with labels containing these words are marked as destructive:

  • delete, remove
  • cancel subscription, unsubscribe
  • pay, checkout, submit order, confirm

Destructive actions require explicit confirmation before execution (when actionsMode is not navigate-only).

extractLinks()

Extracts anchor elements for navigation. Distinguishes between internal links (same origin - indexed as sections) and external links (different origin).

Signature

function extractLinks(html: string, url: string): IndexRecord[]

// Internal links: type: 'section' (can be searched and navigated)
// External links: type: 'link' (navigation targets)
// Link extraction rules:
// - Skips links with rel="nofollow"
// - Skips href="#" and javascript: links
// - Requires non-empty text content
// - Resolves relative URLs against base

// External link detection:
const isExternal = !resolvedUrl.startsWith(window.location.origin);

links.push({
  id: `${url}#link-${links.length}`,
  url: resolvedUrl,
  headingText: linkText,
  headingId: `link-${links.length}`,
  breadcrumb: '',
  bodyText: linkText,
  type: isExternal ? 'link' : 'section',
  selector
});

extractFiles()

Extracts file download links based on file extensions. Recognizes common document, spreadsheet, presentation, and archive formats.

Signature

function extractFiles(
  html: string,
  url: string,
  extensions?: string
): IndexRecord[]

// Default extensions:
// pdf, doc, docx, xls, xlsx, ppt, pptx, zip, csv
// File detection logic:
const fileExtensions = extensions?.split(',').map(e => e.trim().toLowerCase()) ??
  ['pdf', 'doc', 'docx', 'xls', 'xlsx', 'ppt', 'pptx', 'zip', 'csv'];

const isFile = fileExtensions.some(ext =>
  href.toLowerCase().endsWith(`.${ext}`) ||
  href.toLowerCase().endsWith(`.${ext}?`) ||
  href.toLowerCase().endsWith(`.${ext}#`)
);

extractMedia()

Extracts media elements for indexing. Images are indexed via alt text and figure captions. Video and audio elements use title or aria-label attributes.

Signature

function extractMedia(html: string, url: string): IndexRecord[]

// Returns IndexRecord[] with type: 'media'
// Each record includes: selector, transcript (optional)
// Image extraction:
for (const img of images) {
  const alt = img.alt.trim();
  if (!alt) continue; // Skip images without alt text
  
  let caption = '';
  const figure = img.closest('figure');
  if (figure) {
    const figcaption = figure.querySelector('figcaption');
    caption = figcaption?.textContent?.trim() || '';
  }
  
  const textToIndex = caption ? `${alt} ${caption}` : alt;
  // Indexes: "alt text figure caption"
}

// Video/Audio extraction:
for (const element of mediaElements) {
  const title = element.getAttribute('title') || element.getAttribute('aria-label');
  if (!title) continue; // Skip without title/aria-label
  
  // Include transcript references if available
  const tracks = element.querySelectorAll('track[kind="captions"]');
}

extractStructuredData()

Extracts JSON-LD structured data from pages. Supports FAQPage schema for question/answer extraction, and generic structured data for other types.

Signature

function extractStructuredData(html: string, url: string): IndexRecord[]

// Returns IndexRecord[] with type: 'structured'
// FAQ questions are indexed with structuredData: { question, answer }
// FAQ Schema extraction:
const faqItems = data.mainEntity || [];

for (const [index, question] of faqItems.entries()) {
  structured.push({
    id: `${url}#structured-faq-${index}`,
    url: url,
    headingText: question.name,
    headingId: `structured-faq-${index}`,
    breadcrumb: '',
    bodyText: `${question.name} ${answer}`,
    type: 'structured',
    structuredData: { question: question.name, answer }
  });
}

// Generic structured data:
structured.push({
  id: `${url}#structured-${type.toLowerCase()}-${index}`,
  url: url,
  headingText: name || 'Structured Data',
  bodyText: `${name} ${description}`.trim(),
  type: 'structured',
  structuredData: data // Full JSON-LD object
});

Utility Functions

stripTags()

Removes HTML tags from a string, returning plain text.

function stripTags(value: string): string {
  let result = '';
  let inTag = false;
  for (let i = 0; i < value.length; i++) {
    const char = value[i];
    if (char === '<' && !inTag) {
      inTag = true;
    } else if (char === '>' && inTag) {
      inTag = false;
    } else if (!inTag) {
      result += char;
    }
  }
  return result.replace(/\s+/g, ' ').trim();
}

// Example:
// Input: "<h1>Hello <em>World</em></h1>"
// Output: "Hello World"

generateSelector()

Generates a CSS selector path for an element.

function generateSelector(element: Element): string {
  const path = [];
  let current = element;
  
  while (current && current.nodeType === Node.ELEMENT_NODE) {
    let selector = current.tagName.toLowerCase();
    
    if (current.id) {
      selector += `#${current.id}`;
      path.unshift(selector);
      break;
    } else if (current.className) {
      selector += `.${current.className.trim().split(/\s+/).join('.')}`;
    }
    
    // Add nth-child for position-based specificity
    let sibling = current.previousElementSibling;
    let siblingIndex = 1;
    while (sibling) {
      if (sibling.tagName === current.tagName) siblingIndex++;
      sibling = sibling.previousElementSibling;
    }
    if (siblingIndex > 1) {
      selector += `:nth-child(${siblingIndex})`;
    }
    
    path.unshift(selector);
    current = current.parentElement;
  }
  
  return path.join(' > ');
}

// Example: "<div class="container"><button>Click</button></div>"
// Returns: "div.container > button:nth-child(1)"

extractHiddenContent()

Makes hidden content visible for extraction.

function extractHiddenContent(doc: Document): void {
  // Expand collapsed details elements
  doc.querySelectorAll('details').forEach(d => {
    if (!d.hasAttribute('open')) {
      d.setAttribute('data-reef-was-closed', 'true');
    }
    d.setAttribute('open', '');
  });
  
  // Remove aria-hidden restrictions
  doc.querySelectorAll('[aria-hidden="false"]').forEach(el => {
    el.removeAttribute('aria-hidden');
  });
}

// Usage:
const doc = new DOMParser().parseFromString(html, 'text/html');
extractHiddenContent(doc);
const sections = extractSections(html, url);

resolveUrl()

Resolves a URL relative to a base URL.

function resolveUrl(value: string, base: string): string {
  if (!value) return base;
  try {
    return new URL(value, base).toString();
  } catch {
    return value; // Return as-is if invalid
  }
}

// Examples:
resolveUrl('/about', 'https://example.com/docs/');
// Returns: "https://example.com/about"

resolveUrl('page.html', 'https://example.com/docs/');
// Returns: "https://example.com/docs/page.html"

Examples

Extract All Content

Extract all content types from a page.

import {
  extractSections,
  extractActions,
  extractFields,
  extractLinks,
  extractFiles,
  extractMedia,
  extractStructuredData
} from './extraction.js';

const html = document.documentElement.outerHTML;
const url = window.location.href;

const allContent = {
  sections: extractSections(html, url),
  actions: extractActions(html, url, '.no-index'),
  fields: extractFields(html, url),
  links: extractLinks(html, url),
  files: extractFiles(html, url, 'pdf,zip'),
  media: extractMedia(html, url),
  structured: extractStructuredData(html, url)
};

const totalRecords = Object.values(allContent)
  .flat().length;
console.log(`Total: ${totalRecords} records`);

Custom File Extensions

Configure file extraction for specific formats.

import { extractFiles } from './extraction.js';

const config = {
  fileExtensions: 'pdf,mp3,mp4,webm' // Only these types
};

// In Indexer, this becomes:
const files = extractFiles(html, url, config.fileExtensions);

console.log('Found files:', files.map(f => f.headingText));

Scoped Extraction

Limit extraction to a specific page region.

import { extractSections, extractActions } from './extraction.js';

const html = document.documentElement.outerHTML;
const scope = '#main-content'; // Only index main content area

// Extract with scope
const doc = new DOMParser().parseFromString(html, 'text/html');
const scopedElement = doc.querySelector(scope);

if (scopedElement) {
  const scopedHtml = new XMLSerializer().serializeToString(scopedElement);
  const sections = extractSections(scopedHtml, url);
  const actions = extractActions(scopedHtml, url);
  
  console.log('Scoped records:', sections.length + actions.length);
}

Hidden Content Extraction

Include collapsed content in search results.

import { extractSections, extractHiddenContent } from './extraction.js';

const html = document.documentElement.outerHTML;
const doc = new DOMParser().parseFromString(html, 'text/html');

// Make hidden content visible before extraction
extractHiddenContent(doc);

// Now extract includes expanded details
const htmlWithHidden = new XMLSerializer().serializeToString(doc);
const sections = extractSections(htmlWithHidden, url);

// Content from <details> elements will now be searchable
