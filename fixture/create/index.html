Overview

Reef is a client-side library that enables AI agents to search, extract, and act on live DOM content. No servers, no APIs—just pure browser power.

Reef is a client-side library that enables AI agents to search, extract, and act on live DOM content. No servers, no APIs—just pure browser power.

Live DOM Indexing

Index headings, buttons, forms, media, and structured data in real-time from your sitemap.

Agentic Actions

Click, type, submit, and navigate—just like a human. Chainable API for developers.

Low-Code Workflows

Define multi-step automation in JSON/YAML. Perfect for non-developers.

Privacy-First

100% client-side. Your data stays local. No external API calls.

Plug-and-Play

Zero-config, works with any static or dynamic site. Zero build step required.

Keyboard-First

Fast, accessible search modal with Cmd/Ctrl+K shortcut. Full keyboard navigation.

Quick Start

Add Reef to your site with a single script tag:

<script src="reef.min.js"></script>

Press Cmd/Ctrl+K on any page to open the search overlay.

Installation

Script Tag (Recommended)

For most users, simply add the script to your HTML:

<script src="reef.min.js"></script>

Or with configuration:

<script 
  src="reef.min.js"
  data-sitemap="/my-sitemap.xml"
  data-max-pages="100"
  data-primary-color="#43d9c8"
></script>

Headless Mode

For programmatic use without the UI:

// Initialize in headless mode
window.Reef = new ReefSearch({ headless: true });

// Or via script attribute
<script src="reef.min.js" data-headless="true"></script>

NPM Package

For build-time integration:

npm install reef-search

# Import specific modules
import { extractSections, createSearchIndex, addToIndex } from 'reef-search';

# Or use the full bundle
import ReefSearch from 'reef-search';
window.Reef = new ReefSearch();

Configuration Options

Configure Reef via script attributes or programmatic API:

Attribute Default Purpose
data-sitemap /sitemap.xml Sitemap path override
data-max-pages 500 Maximum number of sitemap pages to fetch
data-scope - CSS selector limiting extraction to a specific content root
data-index-actions true Enable indexing of buttons/toggles (also gates field indexing)
data-index-media true Enable indexing of images, video, and audio
data-index-structured-data true Enable indexing of JSON-LD structured data
data-index-hidden true Include collapsed content such as <details> elements
data-file-extensions pdf,doc,docx,... File-link classification
data-actions-mode execute execute or navigate-only — see Safety section
data-hotkey ctrlk,cmdk Custom keyboard shortcut(s)
data-headless false When true, builds index without rendering the modal
data-use-worker-indexing false Offload HTML parsing to a Web Worker for large sites

Use Cases

Web Automation

Automate repetitive tasks like form filling, data entry, and navigation workflows.

const agent = window.Reef.agent();
await agent
  .click("#login-button")
  .type("#email", "user@example.com")
  .type("#password", "secret")
  .submit();

Dynamic Scraping

Extract real-time data from dashboards and dynamic web applications.

const agent = window.Reef.agent();
const data = await agent.extract("#data-table");
console.log(data);

AI Assistants

Build intelligent agents that interact with web apps for customer support, testing, etc.

const workflow = [
  { action: "navigate", url: "/dashboard" },
  { action: "click", selector: "#refresh" },
  { action: "extract", selector: "#data-table" }
];
await window.Reef.executeWorkflow(workflow);

Site Search

Add fast, client-side search to any static or dynamic website.

<script src="reef.min.js"></script>

Press Cmd/Ctrl+K to open search modal.

Low-Code Automation

Enable non-developers to create automation workflows with JSON configuration.

{
  "steps": [
    { "action": "click", "selector": "#login" },
    { "action": "type", "selector": "#email", "value": "test@example.com" },
    { "action": "submit" }
  ],
  "maxRetries": 3,
  "retryDelay": 500
}

Custom Integrations

Integrate Reef into your existing applications with the headless API.

// Custom React component
function SearchComponent() {
  const [results, setResults] = useState([]);
  
  const handleSearch = (query) => {
    setResults(window.Reef.search(query, 10));
  };
  
  return <input onChange={(e) => handleSearch(e.target.value)} />;
}
