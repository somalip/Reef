Overview

The Agent class provides a programmatic API for AI agents to interact with web pages. It offers chainable methods for automation, data extraction, and workflow execution.

Constructor

new Agent(index: SearchIndex, inspector: InspectorInterface, actionsMode?: 'execute' | 'navigate-only')

Creates a new Agent instance.

index SearchIndex

The search index to query for actionable elements.

inspector InspectorInterface

Visual inspector instance for debugging.

actionsMode 'execute' | 'navigate-only' Optional Default: 'execute'

Determines whether destructive actions should be executed or just navigated to.

import { Agent } from './src/agent.js';
import { createSearchIndex } from './src/search-index.js';

// Create dependencies
const index = createSearchIndex();
const inspector = {
  activate: () => {},
  deactivate: () => {},
  isActive: () => false,
  setRecords: () => {}
};

// Create agent
const agent = new Agent(index, inspector, 'execute');

// Or get from ReefSearch
const agent = window.Reef.agent();

Note: In most cases, you should use window.Reef.agent() to get a properly configured Agent instance with access to the current index and configuration.

Action Methods

These methods allow the agent to interact with DOM elements. All methods are async and return Promise<this> for chaining.

click(selector: string | IndexRecord): Promise<this>

Click on an element matching the selector or IndexRecord.

// Click by selector
await agent.click('#login-button');

// Click by IndexRecord
await agent.click(record);

// Chain with other actions
await agent
  .click('#menu-toggle')
  .click('#settings-link')
  .click('#profile-tab');

type(selector: string | IndexRecord, value: string): Promise<this>

Type text into an input, textarea, or select element.

// Type into input
await agent.type('#email', 'user@example.com');

// Type into textarea
await agent.type('#message', 'Hello, World!');

// Chain typing with other actions
await agent
  .type('#username', 'john_doe')
  .type('#password', 'secret123')
  .click('#login-button');

Note: This method properly dispatches input and change events to ensure form validation and React/Vue reactivity work correctly.

submit(selector?: string | IndexRecord): Promise<this>

Submit a form or click a submit button.

// Submit without selector (uses first form)
await agent.submit();

// Submit specific form
await agent.submit('#login-form');

// Submit specific button
await agent.submit('#submit-button');

// Chain with form filling
await agent
  .type('#email', 'user@example.com')
  .type('#password', 'secret')
  .submit();

Navigation Methods

navigate(url: string): Promise<this>

Navigate to a URL.

// Absolute URL
await agent.navigate('https://example.com/dashboard');

// Relative URL
await agent.navigate('/settings');

// Hash navigation
await agent.navigate('#profile-section');

// Chain navigation with actions
await agent
  .navigate('/login')
  .type('#email', 'user@example.com')
  .type('#password', 'secret')
  .submit();

back(): Promise<this>

Navigate back in the browser history.

// Go back
await agent.back();

// Chain with other actions
await agent
  .click('#go-to-page-2')
  .wait(1000)
  .back();

forward(): Promise<this>

Navigate forward in the browser history.

// Go forward
await agent.forward();

// Chain with back
await agent
  .back()
  .wait(500)
  .forward();

wait(timeout?: number): Promise<this>

Wait for a specified amount of time.

// Wait 1 second (default)
await agent.wait();

// Wait 2 seconds
await agent.wait(2000);

// Wait 500ms
await agent.wait(500);

// Chain waiting with actions
await agent
  .click('#refresh')
  .wait(2000) // Wait for data to load
  .extract('#data-table');

Data Methods

extract(selector: string | IndexRecord): Promise<any>

Extract data from an element.

// Extract from input
await agent.type('#email', 'test@example.com');
const emailValue = await agent.extract('#email');
console.log(emailValue); // 'test@example.com'

// Extract text content
await agent.navigate('/about');
const heading = await agent.extract('h1');
console.log(heading); // The h1 text

// Extract from textarea
await agent.type('#message', 'Hello');
const message = await agent.extract('#message');
console.log(message); // 'Hello'

// Extract from select
const selectedOption = await agent.extract('#country');
console.log(selectedOption); // The selected value

findActionable(text: string): Promise<IndexRecord | null>

Find an actionable element (action or field) by text match.

// Find by exact text
const loginButton = await agent.findActionable('Login');
if (loginButton) {
  await agent.click(loginButton);
}

// Find by partial text
const saveButton = await agent.findActionable('Save');
if (saveButton) {
  await agent.click(saveButton);
}

// Use with chaining
await agent
  .findActionable('Submit')
  .then(button => button ? agent.click(button) : Promise.resolve());

Session Methods

getSession(): AgentSession

Get the current session information.

// Get session info
const session = agent.getSession();

// Session object structure:
interface AgentSession {
  id: string;              // Unique session ID
  url: string;            // Current page URL
  timestamp: number;      // Session creation timestamp
  cookies: Record;     // Current cookies
  localStorage: Record; // localStorage snapshot
}

// Example usage
console.log('Session ID:', session.id);
console.log('Current URL:', session.url);
console.log('Cookies:', session.cookies);

Privacy Note: Session information is collected for debugging and automation purposes. Cookies and localStorage data are not transmitted anywhere and remain in the browser.

Workflow Methods

executeWorkflow(steps: WorkflowStep[], options?: WorkflowOptions): Promise<void>

Execute a multi-step workflow with error handling and retry logic.

// Define workflow steps
await agent.executeWorkflow([
  { action: 'navigate', url: '/login' },
  { action: 'type', selector: '#email', value: 'user@example.com' },
  { action: 'type', selector: '#password', value: 'secret123' },
  { action: 'click', selector: '#login-button' },
  { action: 'wait', timeout: 2000 },
  { action: 'extract', selector: 'h1' }
]);

// With options
await agent.executeWorkflow(
  [/* steps */],
  {
    maxRetries: 3,
    retryDelay: 500,
    stopOnError: true,
    onStepStart: (step, index) => {
      console.log(`Starting step ${index}:`, step.action);
    },
    onStepComplete: (step, index, result) => {
      console.log(`Completed step ${index}`, result);
    },
    onStepError: (step, index, error) => {
      console.error(`Step ${index} failed:`, error);
    }
  }
);

Property Type Description
index SearchIndex The search index. Private property.
inspector InspectorInterface Visual inspector instance. Private property.
actionsMode 'execute' | 'navigate-only' Current actions mode. Private property. 

// Inspector Interface
export type InspectorInterface = {
  activate: () => void;
  deactivate: () => void;
  isActive: () => boolean;
  setRecords: (records: IndexRecord[]) => void;
};

// Action Result
export interface ActionResult {
  success: boolean;
  reason?: string;
  element?: Element;
}

// Agent Session
export interface AgentSession {
  id: string;              // Unique session ID
  url: string;            // Current page URL
  timestamp: number;      // Session creation timestamp
  cookies?: Record;     // Current browser cookies
  localStorage?: Record; // Current localStorage data
}

// Workflow Step
export interface WorkflowStep {
  action: 'click' | 'type' | 'navigate' | 'extract' | 'submit' | 'back' | 'forward' | 'wait';
  selector?: string;
  value?: string;
  url?: string;
  recordId?: string;
  timeout?: number;
}

// Workflow Options
export interface WorkflowOptions {
  maxRetries?: number;
  retryDelay?: number;
  stopOnError?: boolean;
  onStepStart?: (step: WorkflowStep, index: number) => void;
  onStepComplete?: (step: WorkflowStep, index: number, result: any) => void;
  onStepError?: (step: WorkflowStep, index: number, error: Error) => void;
}

Examples

Login Workflow

Complete login automation with error handling.

const agent = window.Reef.agent();

const login = async (email, password) => {
  try {
    await agent
      .navigate('/login')
      .type('#email', email)
      .type('#password', password)
      .submit();
    
    // Verify success
    await agent.wait(2000);
    const welcomeText = await agent.extract('.welcome-message');
    
    if (welcomeText.includes('Welcome')) {
      console.log('Login successful!');
      return true;
    }
    
    console.log('Login may have failed');
    return false;
  } catch (error) {
    console.error('Login error:', error);
    return false;
  }
};

login('user@example.com', 'secret123');

Form Data Extraction

Extract data from a form after submission.

const agent = window.Reef.agent();

const extractFormData = async () => {
  // Navigate to form
  await agent.navigate('/user-profile');
  
  // Extract all form fields
  const name = await agent.extract('#name');
  const email = await agent.extract('#email');
  const bio = await agent.extract('#bio');
  const country = await agent.extract('#country');
  
  return {
    name,
    email,
    bio,
    country
  };
};

// Usage
extractFormData().then(data => {
  console.log('Profile data:', data);
});

Multi-Page Workflow

Navigate through multiple pages and extract data.

const agent = window.Reef.agent();

const scrapeDashboard = async () => {
  // Navigate to dashboard
  await agent.navigate('/dashboard');
  
  // Extract summary data
  const stats = await agent.extract('.stats-summary');
  
  // Click to view details
  await agent.click('.view-details');
  await agent.wait(1000);
  
  // Extract detailed data
  const details = await agent.extract('.details-table');
  
  // Go back
  await agent.back();
  
  // Navigate to another section
  await agent.navigate('/dashboard#recent-activity');
  await agent.wait(1000);
  
  const activity = await agent.extract('.activity-list');
  
  return { stats, details, activity };
};

scrapeDashboard().then(data => {
  console.log('Dashboard data:', data);
});

Conditional Actions

Perform actions based on page content.

const agent = window.Reef.agent();

const handleNotification = async () => {
  await agent.navigate('/notifications');
  
  // Check if there are notifications
  const countText = await agent.extract('.notification-count');
  const count = parseInt(countText);
  
  if (count > 0) {
    // Click to mark all as read
    await agent.click('.mark-all-read');
    
    // Or process each notification
    for (let i = 0; i < count; i++) {
      const notification = await agent.extract(`.notification:nth-child(${i + 1})`);
      console.log('Notification:', notification);
      
      await agent.click(`.notification:nth-child(${i + 1}) .mark-read`);
      await agent.wait(200);
    }
  } else {
    console.log('No new notifications');
  }
};

handleNotification();

Workflow with Retries

Execute a workflow with automatic retry on failure.

const agent = window.Reef.agent();

const unstableWorkflow = async () => {
  await agent.executeWorkflow([
    { action: 'navigate', url: '/dashboard' },
    { action: 'click', selector: '#refresh-data' },
    { action: 'wait', timeout: 3000 },
    { action: 'extract', selector: '#chart-data' }
  ], {
    maxRetries: 5,              // Retry up to 5 times
    retryDelay: 1000,          // Wait 1 second between retries
    stopOnError: false,        // Continue with next step even if one fails
    onStepError: (step, index, error) => {
      console.warn(`Step ${index} (${step.action}) failed, retrying...`, error);
    },
    onStepComplete: (step, index, result) => {
      console.log(`Step ${index} (${step.action}) completed`);
    }
  });
};

unstableWorkflow().catch(error => {
  console.error('Workflow failed after retries:', error);
});

Session Information

Access session data for debugging and logging.

const agent = window.Reef.agent();

// Get session info
const session = agent.getSession();

console.log('=== Session Information ===');
console.log('Session ID:', session.id);
console.log('Current URL:', session.url);
console.log('Timestamp:', new Date(session.timestamp).toISOString());
console.log('Cookies:', session.cookies);
console.log('localStorage keys:', Object.keys(session.localStorage || {}));

// Use session ID for logging
const logAction = (action, details = {}) => {
  console.log(`[${session.id}] ${action}`, details);
};

logAction('agent_created');
logAction('navigation_started', { url: '/dashboard' });
