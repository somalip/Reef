Overview

The browser.ts module serves as the entry point for using Reef in browser environments. It creates a global window.Reef instance that can be accessed from anywhere in your page.

Global Instance

Creates a single ReefSearch instance available globally as window.Reef

Automatic Initialization

Automatically initializes when imported, no additional setup required

Script Tag Ready

Can be used via <script> tag without explicit imports

Source Code

browser.ts

The complete source code for the browser module.

/**
 * @file Browser entry point.
 * Instantiates ReefSearch and exposes it globally as `window.Reef`.
 */

import { ReefSearch } from './reef';

const reef = new ReefSearch();
(window as Window & { Reef?: ReefSearch }).Reef = reef;

Usage

As ES Module

Import the browser module directly.

// Import in your application
import './src/browser.js';

// The global instance is now available
window.Reef.open();
window.Reef.search('my query');

Via Script Tag

Load Reef directly in the browser using a script tag.

<!-- In your HTML file -->
<script type="module">
  import './src/browser.js';
</script>

<!-- Or use a bundled version -->
<script type="module" src="./dist/browser.js"></script>

Accessing the Global Instance

Once loaded, access Reef from anywhere.

// From console
Reef.open();
Reef.search('documentation');

// From inline scripts
<script>
  // Reef is available here
  Reef.toggleInspector();
</script>

// From event handlers
<button onclick="Reef.open()">Search</button>

Complete Example

Full HTML page example with Reef integrated.

<!DOCTYPE html>
<html>
<head>
  <title>My App</title>
  <script type="module">
    import './src/browser.js';
  </script>
</head>
<body>
  <h1>My Application</h1>
  
  <button onclick="Reef.open()">
    Open Search (Cmd/Ctrl + K)
  </button>
  
  <button onclick="Reef.toggleInspector()">
    Toggle Inspector
  </button>
  
  <script>
    // Keyboard shortcut
    document.addEventListener('keydown', (e) => {
      if ((e.ctrlKey || e.metaKey) && e.key === 'k') {
        e.preventDefault();
        Reef.open();
      }
    });
  </script>
</body>
</html>
