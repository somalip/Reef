Welcome to this demo documentation site. Press Cmd+K (or Ctrl+K) to search across all pages.

Overview

This site demonstrates Reef Search, a zero-build search overlay for static websites. The search indexes all pages and allows instant searching by section heading.

The indexed content includes heading text, body content, and metadata from each page.

Getting Started

To begin using Reef Search on your site, follow these simple steps:

First, include the script tag in your HTML. The script will automatically discover pages from your sitemap and build an in-memory search index.

No server configuration is needed. Search works entirely in the browser.

Usage

Press Cmd/Ctrl+K to open the search modal. Type your query and results will appear instantly.

You can navigate results with arrow keys and press Enter to visit a page. If the heading has an ID, the page will scroll to that section.
