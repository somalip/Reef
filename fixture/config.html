Configuration

Reef Search can be customized with data attributes on the script tag.

Available Options

  • data-sitemap - Override sitemap path (default: sitemap.xml)
  • data-max-pages - Crawl cap (default: 500)
  • data-scope - CSS selector for indexable root (default: auto-detect)

JavaScript API

Use the global Reef Search object for programmatic control:

Reef Search.open();
Reef Search.reindex();
