Installation

To install Reef Search on your site, add the following script tag to your HTML:

<script src="../dist/Reef.min.js"></script>

Requirements

The search works in all modern browsers. No additional dependencies or build steps are required.

CDN Usage

For production sites, you can load the script from a CDN:

<script src="NOT HERE FOR NOW"></script>

Configuration Options

All configuration is done via data attributes on the script tag.
