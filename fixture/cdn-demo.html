Reef Search CDN Demo

This page demonstrates using Reef Search from a CDN (jsDelivr) instead of hosting the file locally.

Installation Methods

Method 1: IIFE Bundle (Classic)

Use the minified IIFE bundle - auto-initializes when loaded:

<script src="https://cdn.jsdelivr.net/npm/reef-search@0.3.2/dist/reef.min.js"></script>

Method 2: ES Module (Recommended)

Use the ESM version for tree-shaking and programmatic control:

<script type="module">
  import reefSearch from 'https://cdn.jsdelivr.net/npm/reef-search@0.3.2/+esm'
  reefSearch.open()
</script>

Method 3: With Configuration

Configure via global before import:

<script>
  window.ReefConfig = {
    sitemap: '/sitemap.xml',
    maxPages: 100,
    hotkey: 'ctrlk,cmdk'
  }
</script>
<script type="module">
  import reefSearch from 'https://cdn.jsdelivr.net/npm/reef-search@0.3.2/+esm'
  reefSearch.open()
</script>

Method 4: Programmatic Creation

Create instance with config directly:

<script type="module">
  import { createReef } from 'https://cdn.jsdelivr.net/npm/reef-search@0.3.2/+esm'
  
  const reef = createReef({
    sitemap: '/sitemap.xml',
    maxPages: 50,
    mode: 'opaque',
    theme: 'dark'
  })
  
  reef.open()
</script>

Available Configuration Options

  • sitemap - Path to sitemap.xml (default: /sitemap.xml)
  • maxPages - Maximum pages to index (default: 500)
  • scope - CSS selector to limit indexing
  • indexActions - Index buttons/toggles (default: true)
  • indexMedia - Index images/video (default: true)
  • indexStructuredData - Index JSON-LD (default: true)
  • mode - UI mode: regular, opaque, high-contrast
  • theme - Theme: light, dark, auto
  • hotkey - Keyboard shortcut (default: ctrlk,cmdk)
  • headless - Disable UI (default: false)

Test It Out

Try pressing ⌘K (Mac) or Ctrl+K (Windows/Linux) to open the search modal.

Quick Start Code

Copy this to any static page:

<script src="https://cdn.jsdelivr.net/npm/reef-search@0.3.2/dist/reef.min.js"></script>

CDN Version Selection

Use these URLs for different version selection:

  • @latest or @0.3.2 - Specific version
  • @0.3 - Major.minor version
  • @beta - Beta releases
