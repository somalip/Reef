Agentic API Reference

MethodDescription
agent()Returns a chainable Agent instance for click().type().submit() flows
executeWorkflow(steps, options)Runs multi-step JSON/YAML workflows with error handling and retries
act(recordId)Executes an action by record ID, returns {success, reason?}
fillField(recordId, value)Fills a field value programmatically with event dispatch
getAgentTools()Returns all actionable elements for agent tool use
getSession()Returns session snapshot with {id, url, timestamp}
navigate(url)Programmatic page navigation
back() / forward()Browser history navigation
wait(timeout)Pauses execution for async operations
extract(selector)Extracts content value from an element
