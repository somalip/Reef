Install the extension

Load the unpacked package into Chrome and pin it to the toolbar.

  1. 01 Open Chrome and go to chrome://extensions
  2. 02 Enable Developer mode in the top-right corner
  3. 03 Click Load unpacked and select the extension folder
  4. 04 Pin the extension using the 📌 icon on the toolbar
