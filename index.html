Agentic API

A robust set of methods exposed on window.Reef to power browser automation and AI agents safely.

Method Description
agent() Chain actions: click(selector).type(selector, value).submit() for human-like flows
executeWorkflow(workflow, options) Run multi-step JSON/YAML workflows with error handling, retries, and callbacks
act(recordId) Execute actions by record ID — click buttons, toggle controls, trigger actions. Returns {success, reason?}
fillField(recordId, value) Fill input fields programmatically with proper event dispatch
getAgentTools() Get all actionable elements with selectors for agent tool use
getSession() Retrieve session snapshot with {id, url, timestamp} for state management
navigate(url) Programmatic navigation with pending action resolution
back() / forward() Browser history control for agents
wait(timeout) Pause execution for async operations
! Destructive actions (delete, pay, checkout) are gated by actionsMode for safety.

Frequently asked

Does it work with React, Vue, or plain HTML?

Yes. It's a plain script tag, so it works the same regardless of framework. For SPAs, it re-indexes intelligently on route changes.

Do I need a server or database?

No. Indexing and searching both happen in the visitor's browser in a background Web Worker.

What happens to what people search for?

Nothing — queries are never transmitted or logged anywhere. Total privacy by default.

How large is the script?

About 50kb uncompressed (much smaller gzipped), so it won't noticeably affect page load times.
